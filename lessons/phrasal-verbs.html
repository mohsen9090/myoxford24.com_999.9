Phrasal Verbs

Give up = Surrender / Quit • Run out (of) = تمام شدن • Take after = resemble in personality
Get over = Recover • Give away = اهدا کردن (ضمیر بین دو قسمت) → give it away • Make (something) up = داستان ساختن • Put out = Extinguish
Prices went up / have gone up • Run out examples: We have run out of petrol / coffee.
